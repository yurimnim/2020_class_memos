반복문의 종류
1.  for
2. while
3. do~ while

<< for 문을 사용하는 형식>>

 for(초기값; 조건식; 증감식) {
	반복수행할 명령어;
}

<<while문을 사용하는 형식>>

초기값;
while(조건식) {
	반복수행할 명령어;
	증감식; 
}

// 조건식엔 반드시 boolean 이 오도록!
(비교연산자가 많이 쓰임)

<<do ~ while 문을 사용하는 형식>>
 
 do {
	반복실행시킬 명령어;
	증감식;
 } while(조건식) 

<<for in (for each) 반복문>>
배열의 요소만큼 반복실행 시키고자 할 떄 사용

for(변수이름: 배열이름) {
	반복실행시킬 명령어
}

<<break와 continue>>


<<break>>
switch 나 반복문에서 사용되며
가장 가까운 switch 나 반복문을 탈출(벗어남) 하는 용도로 사용 

주의> if 의 {} 탈출하는 개념은 없음 .
if 는 탈출하는 조건을 표현하기 위해서 사용.

<< continue >>
반복문에서만 사용되며 
continue를 만나면 
continue 이후의 명령은 실행하지 않고 

반복문의 조건식이나 증감식을 실행하러감 

***똑같은 문제해결을 할때에 continue를 이용할 수 도 이용 안할수도 있음***
